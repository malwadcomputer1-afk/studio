# **App Name**: Verdant Farm Manager

## Core Features:

- Dashboard Overview: Display key farm stats like attendance, activities, expenses.
- Staff Management: Add, edit, and manage staff member details.
- Attendance Tracking: Track daily staff attendance (present, absent, etc.) with calendar view.
- AI-Powered Salary Calculation: Lovable AI tool to calculate salaries based on attendance, hourly rates, and deductions.
- Payment Tracking: Record salary payments with date, amount, and staff details.
- Daily Activity Logging: Log daily farm activities and link to staff.
- Expense Tracking: Record farm expenses by category, field, and crop.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) for a natural feel.
- Background color: Light cream (#F5F5DC) to create a warm, inviting backdrop.
- Accent color: Warm brown (#A0522D) for highlights and important elements.
- Body and headline font: 'PT Sans' for a modern and readable style.
- Use simple, line-style icons representing farm elements (crops, staff, tools).
- Responsive design for use on desktop and mobile devices in the field.
- Subtle transitions for navigation.
